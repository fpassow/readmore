readmore.js Examples

These are examples for readmore.js, a bit of plain old JavaScript which allows the user to click a "...read more" link to reveal more content. And then they can click "read less" to hide the content again.

The code and a README are at https://github.com/fpassow/readmore

  1. Very simple example
  2. Readless link can be outside of the hidden content
  3. Readmore text can start mid-paragraph
  4. Or put all the readmore content in a single div
  5. Two expandable sections on a page
  6. Example with other display:block elements
